Par défaut, le démontage est temporaire : au redémarrage, Linux remontera ce qui existe encore dans /etc/fstab.
Dossier supprimé seulement s'il est vide, démonté, dans une racine autorisée, et jamais sous /mnt/user ou /mnt/user0.
Coche /etc/fstab seulement si le disque doit rester démonté après reboot.

Modifier le label du volume
Par sécurité, le changement de label est autorisé uniquement quand le volume est démonté.
Limites usuelles : XFS 12 caractères, ext4 16, FAT 11. Utilise un nom simple : lettres, chiffres, tiret, underscore.
Ajouter un dossier MergerFS à une ligne
Ajoute une ou plusieurs sources, puis une destination. La ligne est écrite dans /etc/fstab et montée à chaud.

Vérifier un volume démonté
La vérification est lancée dans une session tmux indépendante de Flask. Le suivi se fait dans le vrai terminal ttyd, pas dans un log Ajax Python.
Sécurité : le backend refuse la vérification si le volume est encore monté ou protégé système/boot.
